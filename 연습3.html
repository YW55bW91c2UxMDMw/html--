CSS 배치 속성

구분선 넣기

h1 { border-bottom: 1px solid black; }

레이아웃

display 속성값에는 block, inline, flow, grid, flex가 있다

주축 정렬 목록

  • flex-start: 시작 부분에 정렬
  • flex-end: 끝에 정렬
  • center: 중앙에 정렬
  • space-between: 박스 각 사이에 공간을 만듬
  • space-around: 사이와 끝 가장자리에도 공간 생김
정렬 예시

body { display: flex; justify-content: space-between; }

교차축 정렬

stretch: 요소의 길이와 교차축의 길이를 같게 함

flex-start: 시작 위치에 정렬

flex-end: 끝 위치에 정렬

center: 중앙에 정렬
